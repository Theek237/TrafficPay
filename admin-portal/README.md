# Admin Portal 📊

[cite_start]This is the Administrative Web Portal for the Sri Lanka Traffic Fine System, allowing senior officials to monitor traffic fine collections nationwide[cite: 29]. [cite_start]It is built as a React Single Page Application (SPA)[cite: 1011].

## 🛠️ Tech Stack
* [cite_start]**Frontend Framework:** React SPA[cite: 1011].
* [cite_start]**Data Visualization:** Recharts / Chart.js[cite: 1010].

## ✨ Key Features
* [cite_start]**National Monitoring:** Provides insights into traffic fine collections across the country[cite: 29].
* [cite_start]**District-wise Collections:** Displays total collections grouped by district[cite: 30].
* [cite_start]**Category Breakdown:** Shows revenue breakdowns by different fine categories[cite: 30].
* [cite_start]**Trend Analysis:** Visualizes daily and monthly collection trends[cite: 923].
* [cite_start]**User Management:** Allows administrators to activate or deactivate user accounts[cite: 970].


## 🚀 Getting Started

### Prerequisites
* Node.js installed on your machine.
* The backend REST API must be running locally.
