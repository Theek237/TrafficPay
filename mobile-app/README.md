# Mobile App 📱

This is the Android mobile application for the Sri Lanka Traffic Fine System. [cite_start]It is designed for drivers to make on-the-spot payments when issued a traffic fine by a police officer[cite: 26, 34]. 

## 🛠️ Tech Stack
* [cite_start]**Frontend Framework:** React Native [cite: 525, 1011] (Expo recommended for fast setup)
* **API Communication:** Axios / Fetch API

## ✨ Key Features
* [cite_start]**Fine Lookup:** Drivers can search for their issued fine using the Reference Number and Category Code[cite: 325, 852].
* [cite_start]**On-the-Spot Payment:** Process payments instantly using the mock payment gateway endpoint[cite: 881].
* [cite_start]**Digital Receipt:** View payment status and receipt details immediately after a successful mock payment[cite: 880, 900].
* [cite_start]**Simplified Location:** Handles fine locations using a simple address string, without requiring complex GPS/location tracking permissions[cite: 656].



## 🚀 Getting Started

### Prerequisites
* Node.js installed on your machine.
* React Native CLI or Expo CLI installed.
* An Android Emulator or a physical Android device for testing.
* The backend REST API must be running locally.
