# Mobile App 📱

This is the Android mobile application for the Sri Lanka Traffic Fine System. It is designed for drivers to make on-the-spot payments when issued a traffic fine by a police officer. 

## 🛠️ Tech Stack
* **Frontend Framework:** React Native  (Expo recommended for fast setup)
* **API Communication:** Axios / Fetch API

## ✨ Key Features
* **Fine Lookup:** Drivers can search for their issued fine using the Reference Number and Category Code.
* **On-the-Spot Payment:** Process payments instantly using the mock payment gateway endpoint.
* **Digital Receipt:** View payment status and receipt details immediately after a successful mock payment.
* **Simplified Location:** Handles fine locations using a simple address string, without requiring complex GPS/location tracking permissions.



## 🚀 Getting Started

### Prerequisites
* Node.js installed on your machine.
* React Native CLI or Expo CLI installed.
* An Android Emulator or a physical Android device for testing.
* The backend REST API must be running locally.
