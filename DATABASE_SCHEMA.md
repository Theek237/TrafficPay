# MongoDB Database Schema

[cite_start]This document outlines the database schema[cite: 816, 817]. 

## ⚙️ Global Standards
* [cite_start]**Database Engine:** MongoDB 7.x with Mongoose 8.x[cite: 823].
* [cite_start]**Naming Convention:** `snake_case` collection names, `camelCase` field names[cite: 823].
* [cite_start]**Auth Tokens:** Single JWT access token with 24-hour expiry[cite: 818, 823].
* [cite_start]**Soft Deletes:** Handled via `isActive: Boolean` flag[cite: 823].
* [cite_start]**Timestamps:** Auto-managed by Mongoose (`{ timestamps: true }`)[cite: 823].

## 4. `fines` Collection
[cite_start]Core entity for issued traffic fines[cite: 932]. [cite_start]GPS fields removed; location handled via an optional string[cite: 934].

| Field | Type | Req | Key | Notes |
| :--- | :--- | :--- | :--- | :--- |
| `_id` | ObjectId | | PK | [cite_start]Auto-generated by MongoDB[cite: 935]. |
| `referenceNo` | String | Yes | UNQ | [cite_start]Pattern: TF-YYYYMMDD-NNNNNN[cite: 935]. |
| `vehicleNo` | String | Yes | IDX | [cite_start]Uppercase (e.g., CAB-1234)[cite: 935]. |
| `officerId` | ObjectId | | FK | [cite_start]Refers to `users._id`[cite: 936]. |
| `categoryId` | ObjectId | Yes | FK | [cite_start]Refers to `fine_categories._id`[cite: 936]. |
| `districtId` | ObjectId | Yes | FK | [cite_start]Refers to `districts._id`[cite: 936]. |
| `amount` | Decimal128| Yes | | [cite_start]Copied from category at issue time[cite: 936]. |
| `status` | String | | IDX | [cite_start]Enum: `PENDING`, `PAID`, `OVERDUE`, `CANCELLED`[cite: 936]. |
| `location.address`| String | | | [cite_start]Optional human-readable string[cite: 936]. |
| `issuedAt` | Date | Yes | | [cite_start]Timestamp of issuance[cite: 936]. |
| `dueDate` | Date | Yes | | [cite_start]`issuedAt` + 30 days[cite: 936]. |

* **Status Flow:** `PENDING` -> `PAID` (on success) or `OVERDUE` (dueDate < now). `OVERDUE` -> `PAID` (late payment). [cite_start]`PAID` and `CANCELLED` are terminal states[cite: 937, 938, 939, 940].
